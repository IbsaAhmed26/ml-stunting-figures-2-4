# Reproducible Figures 2–4 (single clean notebook)

This repository contains **one** clean notebook that reproduces Figures 2–4 for the manuscript.

## Run
Open `analysis/Figures_2_3_4_clean.ipynb` in Jupyter (R kernel) or in an environment that supports R in notebooks.

Required input files are in `data/`.

Outputs are written to `outputs/figures/` (PNG + PDF) and `outputs/session_info.txt`.

## Note on Figure 3
Figure 3 requires a small pipeline-component coding file: `data/pipeline_components_28studies.csv`.
A template is provided as `data/pipeline_components_28studies_TEMPLATE.csv`.
